# Takeshi's SDN 開發筆記

本資料夾下不定期跟新有關我在 SDN Controller 開發上的一箱相關程式，目前有的東西如下：

- Ryu
- mininet topology with python

## How to use

**作業系統**

Ubuntu 14.04 或更高

**需要安裝的工具及軟體**

1. [mininet 2.1.0+][1]
2. [OpenFlow 1.0(optional) / 1.3][2]
3. [python 3][3]

**可裝可不裝的工具/輔助設定**

1. [Sublime Text][4]
2. [Atom][7]
3. [tmux][5]
4. [Mosky's vimrc][6]
